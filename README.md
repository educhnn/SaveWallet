# SaveWallet

SaveWallet es una aplicación de consola en Python para registrar y administrar gastos personales.

El objetivo es poder llevar un control básico de gastos, y crear un proyecto.

## Objetivos
Proyecto práctico para mejorar.
- Lógica de programación en Python
- Manejo de archivos (JSON y CSV)
- Organización y estructuración de código

## Funciones

- Agregar gastos (monto y categoría)
- Ver todos los gastos
- Editar los gastos existentes
- Borrar gastos
- Filtrar por categoría
- Ver el total de los gastos
- Guardado automático en archivo JSON
- Exportar gastos a archivo CSV

---

## Tecnologías

- Python3
- Librerias estandar:
    - json
    - os

---

## Como ejecutar el programa

1. Clona el repositorio o descarga los archivos.
2. En la carpeta del proyecto ejecuta: "main.py".
